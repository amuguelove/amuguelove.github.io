Java基础类型占用字节数

发表于 November 6, 2018
标签: Java  

在Java中一共有8种基本数据类型，其中有4种整型，2种浮点型，1种用于表示Unicode编码的字符但愿的字符类型和一种表示真值的boolean类型。

### 整型

类型 存储大小(字节) Bit数 取值范围 备注
int 4 4 * 8
short 2 2 * 8
long 8 8 * 8
byte 1 1 * 8 -128～127

浮点型

类型 存储大小（字节） Bit数 取值范围 备注
float 4 4 * 8 需要在数字末尾在f或F（如：3.14f）
double 8 4 * 8 Java浮点数默认类型

char类型

类型 存储大小（字节） Bit数 取值范围 备注
char 2 2 * 8

### boolean类型

类型 存储大小（字节） Bit数 取值范围 备注
boolean 1 1 * 8 false/true
