Mac下安装Golang并配置环境

发表于 December 18, 2018
标签: Golang   MacOS  

在MacOS上主要有如下两种方法安装

一、HomeBrew

homebrew是Mac系统下面目前使用最多的管理软件的工具，目前已支持Go，可以通过命令直接安装Go。

brew update && brew upgrade
brew install go

这样安装之后通过命令行输入go就可以看到相关的信息。输入go env查看环境信息。

二、pkg文件

直接去官方下载安装包，然后双击安装，之后同样地输入go、go env、go version等查看是否安装。

三、环境变量配置

1. 是否存在环境变量配置文件

vi ~/.zshrc

如果不存在，则新建。我本地使用的shell是zsh，如果你的是bash，则编辑~/.bash_profile文件即可，没有先创建。

2. 添加go环境变量

# Golang
export GOROOT=/usr/local/go
export GOPATH=/Users/bo.liang/Documents/go_workspace
export PATH=$PATH:$GOROOT/bin:$GOPATH/bin

3. 立即生效

source ~/.zshrc

4. 查看

$ go env

到这里，你就配置好golang的环境了。
